# L'echappee d'Emma

Site vitrine pour L'echappee d'Emma, une agence de voyage sur mesure a Sucy-en-Brie. Le projet presente les offres, l'approche personnalisee, les destinations et les moyens de contact.

## Technologies

- Vite + React + TypeScript
- Tailwind CSS et shadcn/ui (Radix UI)
- React Router
- React Hook Form + Zod
- Framer Motion
- Vitest (tests)

## Demarrage rapide

Prerequis : Bun (recommande) ou Node.js + npm.

```sh
# Installation des dependances
bun install

# Lancer le serveur de developpement
bun dev
```

Equivalent npm :

```sh
npm install
npm run dev
```

## Scripts utiles

- `bun dev` : serveur de developpement
- `bun build` : build de production
- `bun preview` : previsualisation du build
- `bun test` : tests unitaires

## Structure rapide

- `src/pages` : pages principales
- `src/components` : composants reutilisables
- `src/assets` : images et medias
